loadjar里边放的是需要导入的低版本的jar包和剩余的所有包，不兼容的话将低版本的包导入进去
springDM里边放的是SpringDM框架需要引入的包
word中是此次学习的总结及经验

由于此处使用的SpringDM为低版本，导致对应需要导入的jar包也只能为低版本的
jdk版本为1.6
